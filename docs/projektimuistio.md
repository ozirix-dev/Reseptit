# Projektimuistio

## Kielisaanto

Tassa projektissa kaytetaan teknisissa asioissa suomen kielta ilman aakkosia.

Tama koskee erityisesti:

- hakemistorakennetta
- tiedostonimia
- skripteja
- teknista dokumentaatiota
- metatietokenttia
- commit-viesteja

Ei kayteta merkkeja:

- `ä`
- `Ä`
- `ö`
- `Ö`

Poikkeus:

- suoraan resepteihin liittyva loppukayttajalle nakyva sisalto saa sisaltaa aakkosia
